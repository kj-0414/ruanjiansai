# 人才职位智能匹配与能力图谱系统 - UI/UX 设计规范

## 1. 设计概述

### 1.1 产品定位
面向求职者与企业的智能人才匹配平台，通过AI技术分析简历与岗位需求，实现精准匹配与能力可视化。

### 1.2 设计目标
- **专业化**：体现AI技术与人力资源领域的专业性
- **易用性**：降低操作门槛，提升任务完成效率
- **视觉吸引力**：采用现代设计语言，增强用户信任感
- **响应式**：适配桌面端与移动端，确保一致体验

### 1.3 目标用户
| 角色 | 核心需求 | 使用场景 |
|------|---------|---------|
| 求职者 | 快速找到匹配岗位、跟踪投递状态、了解自身能力画像 | 移动端为主，碎片化使用 |
| 企业HR | 高效筛选候选人、管理岗位发布、查看匹配结果 | 桌面端为主，集中处理 |

## 4. 组件设计

### 4.1 按钮系统
| 类型 | 样式 | 用途 |
|------|------|------|
| Primary | 实心底色 + 白色文字 | 主要操作（提交、确认） |
| Secondary | 透明背景 + 描边 | 次要操作（取消、返回） |
| Ghost | 透明背景 + 文字链接 | 低优先级操作 |
| Icon | 仅图标 | 工具栏操作 |
| FAB | 圆形悬浮按钮 | 核心操作（如新建、上传） |

| 尺寸 | 高度 | 内边距 | 字号 | 圆角 |

| Medium | 40px | 12px 24px | 14px | 10px |
| Small | 32px | 8px 16px | 13px | 8px |

## 6. 交互模式

### 6.1 动画曲线
```scss
--ease-out: cubic-bezier(0.25, 0.1, 0.25, 1);
--ease-in: cubic-bezier(0.42, 0, 1, 1);
--ease-in-out: cubic-bezier(0.42, 0, 0.58, 1);
--ease-bounce: cubic-bezier(0.68, -0.55, 0.265, 1.55);
```

### 6.2 动画时长
| 场景 | 时长 | 曲线 |
|------|------|------|
| 微交互 | 150ms | ease-out |
| 状态切换 | 200ms | ease-in-out |
| 页面过渡 | 300ms | ease-in-out |
| 复杂动画 | 500ms | ease-bounce |

### 6.3 加载状态
- **骨架屏**: 用于列表、卡片加载
- **进度条**: 用于文件上传、长操作
- **Loading Spinner**: 用于按钮、弹窗

### 6.4 空状态设计
- 插图: 80x80px 线性图标
- 标题: 16px 主色
- 描述: 14px 辅助文本
- 操作按钮: 引导用户下一步操作

## 8. 页面设计

### 8.1 登录页面
**设计要点:**
- 全屏渐变背景 (主色到AI紫色)
- 居中卡片式表单，带微投影
- 品牌Logo + 系统名称顶部展示
- 登录/注册切换带动画过渡
- 输入框聚焦状态高亮
- 密码可见性切换图标
- 手机号验证码倒计时动画

### 8.2 主布局
**顶部导航栏:**
- 高度: 64px
- 背景: 白色 + 底部阴影
- 左侧: 系统Logo + 名称
- 右侧: 通知图标 + 用户头像 + 角色切换
- 移动端: 汉堡菜单图标

**侧边导航栏:**
- 宽度: 280px (展开) / 72px (收起)
- 背景: 白色 + 右侧边框
- 菜单项高度: 48px
- 激活状态: 左侧蓝色指示条 + 主色背景
- 图标 + 文字，hover时背景变化
- 支持折叠/展开动画

**内容区域:**
- 背景: `#F5F5F7`
- 内边距: 24px
- 最大宽度: 1440px (居中)

### 8.3 首页设计
**统计卡片区域:**
- 3列栅格布局
- 每张卡片: 图标 + 数值 + 趋势 + 标题
- 悬停时卡片上移 + 阴影加深
- 渐变色背景图标容器
- 数值字体: 32px, Bold

**数据可视化区域:**
- ECharts 图表容器
- 圆角卡片包裹
- 标题区 + 图表区
- 支持切换时间范围

**最近匹配记录:**
- 表格形式展示
- 匹配分数使用渐变色标签
- 行hover效果
- 点击行查看详情

### 8.4 岗位列表页
**搜索与筛选区:**
- 顶部固定搜索栏
- 筛选条件以标签形式展示
- 支持多条件组合筛选
- 筛选面板滑入动画

**岗位卡片:**
- 列表/网格视图切换
- 卡片包含: 职位名称、薪资、公司、地点、技能标签
- 悬停显示操作按钮
- 匹配度标识 (仅匹配后)
- 收藏按钮 (心形图标)

### 8.5 简历上传页
**上传区域:**
- 拖拽上传框，带虚线边框
- 支持 .docx / .pdf 格式
- 上传进度条
- 上传成功后显示解析结果预览

**解析结果:**
- 结构化展示 (教育、技能、经验)
- 可编辑字段
- 确认提交按钮

## 9. 设计原则

### 9.1 一致性
- 相同功能使用相同交互模式
- 全局统一的色彩、字体、间距
- 组件库复用，避免重复造轮子

### 9.2 反馈性
- 操作即时反馈 (按钮按下、加载状态)
- 成功/失败明确提示 (Toast消息)
- 状态可视化 (进度条、骨架屏)

### 9.3 效率性
- 核心操作3步内完成
- 常用功能快捷入口
- 表单自动填充与智能提示

### 9.4 容错性
- 操作前确认 (危险操作)
- 错误提示明确可理解
- 支持撤销/回退操作

## 11. 无障碍设计

### 11.1 对比度
- 文本与背景对比度 ≥ 4.5:1 (WCAG AA)
- 大文本对比度 ≥ 3:1

### 11.2 键盘导航
- 所有交互元素支持 Tab 键访问
- 激活状态使用 Enter/Space 键
- 焦点指示器清晰可见

### 11.3 屏幕阅读器
- 所有图标按钮带 `aria-label`
- 表单输入关联 `label`
- 动态内容使用 `aria-live`
