# 人才职位智能匹配与能力图谱系统使用手册

## 一、系统简介

本系统是一个基于AI的人才职位智能匹配与能力图谱系统，旨在帮助企业和求职者更高效地匹配职位和人才。系统支持简历上传、岗位发布、AI智能匹配、能力图谱可视化等功能。

## 二、快速开始

### 步骤1：启动系统

1. 找到项目根目录下的 `start_all.bat` 文件
2. 双击运行该文件
3. 系统会自动启动后端服务和前端服务
4. 启动完成后，浏览器会自动打开前端页面

### 步骤2：使用系统

1. 在登录页面输入用户名和密码进行登录
   - 初始测试账号：admin / 123456
   - 角色选择：求职者或企业

2. 登录成功后，您可以：
   - 上传简历（求职者）
   - 发布岗位（企业）
   - 进行岗位匹配
   - 查看能力图谱

### 步骤3：退出系统

点击页面右上角的"退出"按钮即可退出系统。

## 三、功能说明

### 1. 登录/注册

- **登录**：输入用户名和密码，选择角色后点击"登录"按钮
- **注册**：输入用户名、密码和角色后点击"注册"按钮

### 2. 简历上传

- 点击左侧导航栏的"简历上传"选项
- 输入简历名称
- 选择PDF或DOCX格式的简历文件
- 点击"上传简历"按钮
- 上传成功后，可以在"我的简历"列表中查看

### 3. 岗位匹配

- 点击左侧导航栏的"岗位匹配"选项
- 选择要匹配的简历和岗位
- 点击"开始匹配"按钮
- 系统会显示匹配结果，包括匹配度、匹配技能、缺失技能和能力图谱

### 4. 首页

- 显示统计数据，包括简历数、岗位数和匹配记录数
- 显示最近的匹配记录

## 四、常见问题排查

### 1. 端口占用

- **问题**：启动服务时提示端口被占用
- **解决方法**：检查是否有其他程序占用了8000（后端）或5173（前端）端口，关闭占用端口的程序后重新启动

### 2. 依赖安装失败

- **问题**：启动服务时提示依赖安装失败
- **解决方法**：检查网络连接是否正常，或手动运行 `pip install -r requirements.txt`（后端）或 `npm install`（前端）

### 3. 页面空白

- **问题**：浏览器打开后页面空白
- **解决方法**：检查前端服务是否正常启动，或刷新浏览器页面

### 4. 登录失败

- **问题**：登录时提示用户名或密码错误
- **解决方法**：检查用户名和密码是否正确，或先注册一个新账号

## 五、系统要求

- **操作系统**：Windows、Linux（支持银河麒麟系统）
- **浏览器**：Chrome、Firefox、Edge等现代浏览器
- **网络**：需要网络连接以访问系统
